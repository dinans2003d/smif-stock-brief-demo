# SMIF Stock Brief Demo

## Overview
This project is a demo dashboard designed to help BAM SMIF analysts quickly see what changed for their assigned stocks and identify which approved news sources to read first.

The tool focuses on **signal awareness and reading prioritization**, not decision-making.

## What this tool does
- Displays stock-level KPIs that highlight material daily changes
- Flags unusual movements using transparent, rule-based thresholds
- Surfaces up to two relevant headlines from approved sources (CNBC, WSJ, FT, Bloomberg)
- Supports analyst-led judgment and deeper reading

## What this tool does NOT do
- Does not make buy/sell or hold recommendations
- Does not summarize or replace full article reading
- Does not provide trading advice or portfolio allocation decisions
